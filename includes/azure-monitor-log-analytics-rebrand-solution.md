>[!NOTE]
>As versões anteriores deste artigo referentes ao Log Analytics como seu próprio serviço. Sua funcionalidade não foi alterada, mas foi renomeada para o recurso de log do Azure Monitor. Com essa renomeação, este artigo descreve os dados que estão sendo armazenados em um espaço de trabalho do Log Analytics no Azure Monitor. Para obter mais informações, confira [Alterações de marca do Azure Monitor](../articles/azure-monitor/azure-monitor-rebrand.md).
