## <a name="delete-anchors"></a>Excluir as âncoras

Para excluir uma âncora de nuvem espacial, você deve usar o `DeleteAnchor()` método. Excluir as âncoras quando não for mais usado é uma boa prática para incluir no início no processo de desenvolvimento e práticas recomendadas. Isso mantém os recursos limpos. Além disso, o mesmo que você possa usar âncoras espacial do Azure gratuitamente, preços podem mudar no futuro. Você pode encontrar detalhes sobre preços na Versão Prévia em nossa [página de preços](https://azure.microsoft.com/pricing/details/spatial-anchors/).
