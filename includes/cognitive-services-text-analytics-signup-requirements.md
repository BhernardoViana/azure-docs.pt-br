É necessário ter uma [assinatura da API de Serviços Cognitivos](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) com acesso à API de Análise de Texto. Se você não tiver uma assinatura, [crie uma conta](https://azure.microsoft.com/try/cognitive-services/?api=bing-web-search-api) gratuitamente. Antes de continuar, você precisará da chave de assinatura de Análise de Texto fornecida depois de ativar sua conta.
