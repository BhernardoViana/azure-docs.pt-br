É necessário ter uma [conta da API dos Serviços Cognitivos](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) com acesso à API de Verificação Ortográfica do Bing. Caso não tenha uma assinatura do Azure, [crie uma conta](https://azure.microsoft.com/try/cognitive-services/?api=text-analytics) gratuitamente. Antes de continuar, você precisará da chave de acesso [fornecida após a ativação da avaliação gratuita](https://azure.microsoft.com/try/cognitive-services/my-apis/) ou de uma chave de assinatura paga do painel do Azure.
