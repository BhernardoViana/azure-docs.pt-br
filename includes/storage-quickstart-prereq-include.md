Para acessar o Armazenamento do Azure, você precisará de uma assinatura do Azure. Se você ainda não tiver uma assinatura, crie uma [conta gratuita](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) antes de começar.

Todo o acesso ao Armazenamento do Azure ocorre por meio de uma conta de armazenamento. Se você ainda não tiver uma conta de armazenamento, crie uma facilmente usando o [portal do Azure](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). Você também pode criar uma conta de armazenamento usando o Azure PowerShell ou a CLI do Azure. Para obter ajuda sobre como criar a conta, confira [Criar uma conta de armazenamento](../articles/storage/common/storage-quickstart-create-account.md).
