> [!NOTE]
> Para obter melhores resultados ao usar a API do Detector de anomalias, seus dados de série de tempo formatada em JSON devem incluir:
> * pontos de dados separados pelo mesmo intervalo, com não mais do que 10% do número esperado de pontos ausentes.
> * pontos de dados pelo menos 12 se seus dados não tiverem um padrão claro sazonal.
> * pelo menos 4 ocorrências padrão se os dados tiverem um padrão claro sazonal.
