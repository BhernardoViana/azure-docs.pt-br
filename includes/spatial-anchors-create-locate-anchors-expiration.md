## <a name="set-expiration"></a>Definir a validade

Também é possível configurar a âncora para expirar automaticamente em uma determinada data no futuro. Quando uma âncora expira, ele não será localizado ou atualizado. Expiração só pode ser definida quando a âncora é criada. Não é possível atualizar posteriormente para expiração. Assim, você pode definir a expiração antes de salvá-lo para a nuvem.
