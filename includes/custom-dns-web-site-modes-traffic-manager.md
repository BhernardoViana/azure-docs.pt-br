A configuração de um nome de domínio personalizado em um aplicativo Web integrado ao Gerenciador de Tráfego só está disponível para o tipo de preço **Standard**.  

Para saber mais sobre os tipos de preço do Serviço de Aplicativo, inclusive sobre como alterar o tipo de preço do seu aplicativo, consulte [Escalar verticalmente um aplicativo no Azure](../articles/app-service/manage-scale-up.md).
