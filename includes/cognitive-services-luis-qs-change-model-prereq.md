* Importe o [aplicativo TravelAgent](https://github.com/Azure-Samples/cognitive-services-language-understanding/blob/master/documentation-samples/quickstarts/change-model/TravelAgent.json) do repositório GitHub cognitive-services-language-understanding. 
* A [**ID do aplicativo**](../articles/cognitive-services/luis/luis-get-started-create-app.md) LUIS para o aplicativo TravelAgent importado. A ID do aplicativo é mostrada no painel do aplicativo. 
* O arquivo **[utterances.json](https://github.com/Azure-Samples/cognitive-services-language-understanding/blob/master/documentation-samples/quickstarts/change-model/utterances.json)** que contém os enunciados de exemplo a serem importadas. 
* O **ID da versão** no aplicativo que recebe os enunciados. A ID padrão é “0.1”.
