Quando você usa o Gerenciador de Tráfego do Microsoft Azure para carregar o tráfego equilibrado no site do Azure, esse site pode ser acessado usando o nome de domínio **\*.trafficmanager.net** atribuído pelo Azure. Você também pode associar um nome de domínio personalizado, como www\.contoso.com, com o seu site para fornecer um nome de domínio mais reconhecível para seus usuários.
