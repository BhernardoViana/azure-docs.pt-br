## <a name="set-up-authentication"></a>Configurar a autenticação

Para acessar o serviço, você precisa fornecer uma chave de conta, o token de acesso ou o token de autenticação do AAD.

### <a name="account-keys"></a>Chaves da conta

Chaves da conta são uma credencial que permite que seu aplicativo para autenticar com o serviço âncoras espacial do Azure. A finalidade das chaves de conta é para ajudá-lo a começar rapidamente. Especialmente durante a fase de desenvolvimento de integração do seu aplicativo com âncoras espacial do Azure. Como tal, você pode usar as chaves da conta, inserindo-os em seus aplicativos cliente durante o desenvolvimento. Conforme você progride além do desenvolvimento, é altamente recomendável mover para um mecanismo de autenticação que é o nível de produção, compatíveis com Tokens de acesso ou autenticação de usuário baseada no AAD. Para obter uma chave de conta para o desenvolvimento, visite sua conta âncoras espacial do Azure e navegue até a guia "Chaves".
