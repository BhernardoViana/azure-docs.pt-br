Você deve ter uma [conta de API de serviços Cognitivos](../articles/cognitive-services/cognitive-services-apis-create-account.md) com acesso à API de reconhecimento de tinta. Caso não tenha uma assinatura do Azure, [crie uma conta](https://azure.microsoft.com/try/cognitive-services/) gratuitamente. Você pode obter sua chave de assinatura do [portal do Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md#access-your-resource) depois de criar sua conta ou do [site do Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) depois de ativar uma avaliação gratuita.
