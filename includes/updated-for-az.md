> [!NOTE]
>
> Este artigo foi atualizado recentemente para usar comandos do novo módulo Az do Azure PowerShell. Esse módulo é o novo módulo do PowerShell para o Azure, compatível com o PowerShell versões 5.x e 6. O AzureRM continuará obtendo atualizações críticas de correção de bug, mas os novos recursos estarão apenas no módulo Az.
>
> * Para saber mais sobre o módulo Az e o suporte contínuo para o módulo AzureRM, confira [Apresentando o novo módulo Az do Azure PowerShell](/powershell/azure/new-azureps-module-az).
> * O módulo Az já está disponível no Cloud Shell, com a compatibilidade com o AzureRM habilitada automaticamente.
>   Para obter instruções sobre como instalar o módulo Az, confira [Instalar o Azure PowerShell](/powershell/azure/install-az-ps).
> * Para habilitar os comandos de compatibilidade com o AzureRM, execute `Enable-AzureRmAlias` após a instalação.
>   Para obter mais informações sobre compatibilidade, confira [Migrar do AzureRM para o Az do Azure PowerShell](/powershell/azure/migrate-from-azurerm-to-az).
