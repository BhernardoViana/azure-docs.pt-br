FedEx, UPS ou DHL podem ser usados para enviar o pacote para datacenter do Microsoft Azure. Se você quiser usar uma operadora que não seja FedEx/DHL, contate a equipe de operações de caixa de dados do Azure em `adbops@microsoft.com`

- Forneça um número de conta da transportadora FedEx, UPS ou DHL válido que a Microsoft usará para enviar as unidades de volta. 
    
    - Um número de conta FedEx, UPS ou DHL é necessário para enviar de volta as unidades de locais nos EUA e na Europa. 
    - Um número de conta DHL é necessário para enviar de volta as unidades de locais na Ásia e na Austrália. 
    - Caso não tenha um número de conta, crie uma conta da operadora [FedEx](http://www.fedex.com/us/oadr/) ou [DHL](http://www.dhl.com/).
- Ao enviar seus pacotes, siga os [Termos de Serviço do Microsoft Azure](https://azure.microsoft.com/support/legal/services-terms/).
- Embale os discos corretamente para evitar possíveis danos e atrasos no processamento.
