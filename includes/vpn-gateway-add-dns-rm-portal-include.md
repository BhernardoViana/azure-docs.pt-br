1. Na seção **Configurações** da página de rede virtual, selecione **Servidores DNS** para abrir a página **Servidores DNS**.

2. Na página **Servidores DNS**, preencha os seguintes valores:

   - **Servidores DNS:** selecione **Personalizado**.

   - **Adicionar servidor DNS:** Insira o endereço IP do servidor DNS que você deseja usar para a resolução de nome.

3. Quando você terminar de adicionar servidores DNS, selecione **Salvar**.

    ![Especificar um servidor DNS](./media/vpn-gateway-add-dns-rm-portal-include/specify-dns-server.png "Especificar um Servidor DNS")
