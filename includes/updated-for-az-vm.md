> Este artigo foi atualizado para usar o novo [módulo Az do Azure PowerShell](/powershell/azure/new-azureps-module-az). O AzureRM continuará obtendo atualizações críticas de correção de bug, mas os novos recursos estarão apenas no módulo Az.
> * O módulo Az já está disponível no Cloud Shell, com a compatibilidade com o AzureRM habilitada.
> * Para instalar ou atualizar, confira [Instalar o Azure PowerShell](/powershell/azure/install-az-ps).
> * Se você instalar o módulo de Az localmente, é possível habilitar a compatibilidade do AzureRM executando `Enable-AzureRmAlias`.
