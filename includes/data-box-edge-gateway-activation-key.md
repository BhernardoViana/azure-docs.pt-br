A chave de ativação que você usa:

- É um Azure Active Directory (Azure AD) baseado em chave de autenticação.
- Expira após três dias.
- Não é usada após a ativação do dispositivo.

Depois de ativar um dispositivo, ele utilizará tokens para se comunicar com o Azure.
