Os requisitos a seguir se aplicam à largura de banda mínima da Internet necessária para seus dispositivos.

- Seu dispositivo tem uma largura de banda de Internet de 20 Mbps dedicada (ou mais) disponível o tempo todo. Essa largura de banda não deve ser compartilhada com nenhum outro aplicativo.
- O dispositivo terá uma largura de banda de Internet de 64 Mbps dedicada (ou mais) ao usar a aceleração de rede.
