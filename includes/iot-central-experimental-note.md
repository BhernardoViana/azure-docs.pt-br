> [!Note]
> O Azure IoT Central atualmente está apresentando um novo padrão de interação entre os dispositivos e modelos de dispositivo. Esse novo design de interface do usuário separa os dispositivos e modelos de dispositivo, o que permite interagir com cada um de forma independente. Se você vir um botão **Modelos de dispositivo** no menu de navegação à esquerda, esse é o novo design de interface do usuário:
