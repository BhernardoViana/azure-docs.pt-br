### <a name="retrieve-host-name-ports-and-access-keys-by-using-the-azure-portal"></a>Recuperar o nome do host, portas e chaves de acesso usando o portal do Azure

Ao se conectar a uma instância do Cache Redis do Azure, os clientes de cache precisam do nome de host, das portas e uma chave para o cache. Alguns clientes podem se referir a esses itens por nomes um pouco diferentes. Você pode recuperar essas informações no portal do Azure.

#### <a name="to-retrieve-the-access-keys-and-host-name"></a>Para recuperar as chaves de acesso e nome do host

1. Para recuperar as chaves de acesso usando o [portal do Azure](https://portal.azure.com), navegue até seu cache e selecione **Chaves de acesso**. 

    ![Chaves do Cache Redis do Azure](media/redis-cache-access-keys/redis-cache-keys.png)

2. Para recuperar o nome de host e as portas, selecione **Propriedades**.

    ![Propriedades do Cache Redis do Azure](media/redis-cache-access-keys/redis-cache-hostname-ports.png)
