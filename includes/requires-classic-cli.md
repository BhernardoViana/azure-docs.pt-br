> [!IMPORTANT]
> Este artigo apresenta conteúdo que requer a CLI clássica do Azure. A versão atual da CLI do Azure não tem suporte para os recursos descritos neste artigo; portanto, a CLI clássica é obrigatória. 
> 
> A CLI clássica pode ser instalado lado a lado com a CLI do Azure modernos, mas é recomendável que a CLI do Azure para todos os novos scripts e implantações.
> Para instalar a CLI clássica, confira [Instalar a CLI clássica do Azure](/cli/azure/install-classic-cli). 
> Para instalar a versão mais recente da CLI, consulte [instalar a CLI do Azure](/cli/azure/install-azure-cli).
