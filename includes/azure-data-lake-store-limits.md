O Azure Data Lake Storage Gen1 é um repositório de hiperescala em toda a empresa para cargas de trabalho analíticas de big data. Você pode usar o Data Lake armazenamento Gen1 para capturar dados de qualquer tamanho, tipo e velocidade de ingestão em um único lugar para análises operacionais e EXPLORATÓRIAS. Não há nenhum limite para a quantidade de dados que podem ser armazenados em uma conta do Data Lake armazenamento Gen1.

| **Recurso** | **Limite padrão** | **Comentários** |
| --- | --- | --- |
| Número máximo de contas do Data Lake armazenamento Gen1, por assinatura por região |10 | Para solicitar um aumento desse limite, entre em contato com o suporte. |
| Número máximo de ACLs de acesso, por arquivo ou pasta |32 | Este é um limite rígido. Use grupos para gerenciar o acesso com menos entradas. |
| Número máximo de ACLs padrão, por arquivo ou pasta |32 | Este é um limite rígido. Use grupos para gerenciar o acesso com menos entradas. |
