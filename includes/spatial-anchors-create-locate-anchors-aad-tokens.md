### <a name="azure-active-directory-authentication"></a>Autenticação do Azure Active Directory

Âncoras espacial do Azure também permite que aplicativos se autentiquem com tokens de Azure Active Directory do usuário. Por exemplo, você pode usar tokens do AAD para integrar com o ASA. Se uma empresa mantém os usuários no AAD, você pode fornecer um token de usuário do AAD no SDK do Azure espacial âncoras. Isso permite que você seja autenticado diretamente o serviço ASA para uma conta que faz parte do mesmo locatário do AAD.
