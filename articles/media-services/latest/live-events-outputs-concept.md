# <a name="live-events-and-live-outputs"></a>Eventos ao Vivo e Saídas Dinâmicas

O Azure Media Services permite entregar eventos ao vivo para seus clientes na nuvem do Azure. Para configurar os eventos de transmissão ao vivo nos Serviços de Mídia v3, é necessário saber mais sobre [Eventos ao Vivo](https://docs.microsoft.com/rest/api/media/liveevents) e [Saídas Dinâmicas](https://docs.microsoft.com/rest/api/media/liveoutputs).
