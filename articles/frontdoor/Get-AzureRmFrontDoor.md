# Get-AzureRmFrontDoor
Obter a configuração do Front Door

## SINTAXE

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzureRmFrontDoor** obtém todos os Front Doors existentes na assinatura atual que correspondam às informações fornecidas.
