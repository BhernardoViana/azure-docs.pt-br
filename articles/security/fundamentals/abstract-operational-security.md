# <a name="azure-operational-security"></a>Segurança Operacional do Azure
## <a name="abstract"></a>Resumo
A segurança operacional do Microsoft Azure refere-se aos serviços, controles e recursos disponíveis aos usuários para proteger seus dados, aplicativos e outros ativos no Azure. A segurança operacional do Azure se baseia em uma estrutura que incorpora o conhecimento adquirido por vários recursos exclusivos da Microsoft, incluindo o SDL (Security Development Lifecycle) da Microsoft, o programa Microsoft Security Response Center e o profundo conhecimento do cenário de ameaças de segurança cibernética. Este white paper descreve como você pode abordar a segurança operacional usando o Azure. Ele aborda vários serviços do Azure, incluindo:
* Logs do Azure Monitor
* Backup do Azure
* Central de Segurança do Azure
* Azure Monitor
* Observador de Rede do Azure
* Análise do Armazenamento do Azure
* Active Directory do Azure
