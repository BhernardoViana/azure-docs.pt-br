# <a name="threat-modeling-tool-releases"></a>Versões da Threat Modeling Tool

A Microsoft Threat Modeling Tool está liberada no momento como um aplicativo gratuito de [clique para baixar](https://aka.ms/threatmodelingtool) para Windows. Esse mecanismo de entrega permite enviar as melhorias e as correções de bug mais recentes por push aos clientes toda vez que eles abrem a ferramenta, facilitando a manutenção e o uso.

## <a name="system-requirements"></a>Requisitos do Sistema

- Sistemas operacionais com suporte
  - Microsoft Windows 10
- Requisitos adicionais
  - Uma conexão de Internet é necessária para receber as atualizações da ferramenta e também os modelos.
