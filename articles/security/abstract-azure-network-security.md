# <a name="azure-network-security"></a>Segurança de rede do Azure
## <a name="abstract"></a>Resumo
Os serviços de rede do Azure maximizam a flexibilidade, disponibilidade, resiliência, segurança e integridade por design. Este white paper fornece detalhes sobre as funções de rede do Azure. Também descreve como os clientes podem usar os recursos de segurança nativos no Azure para ajudar a proteger seus ativos de informações.
Os públicos-alvo deste white paper incluem:
- Gerentes técnicos, administradores de rede e desenvolvedores que estão procurando soluções de segurança disponíveis e com suporte no Azure.
- SMEs ou executivos de processo de negócios que desejam obter uma visão geral de alto nível de tecnologias de rede do Azure e serviços que são relevantes para segurança de rede na nuvem pública do Azure.
