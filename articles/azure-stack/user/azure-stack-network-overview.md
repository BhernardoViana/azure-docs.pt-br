# <a name="introduction-to-azure-stack-networking"></a>Introdução à rede do Azure Stack

*Aplica-se a: integrados do Azure Stack, sistemas e o Kit de desenvolvimento do Azure Stack*

O Azure Stack oferece uma variedade de recursos de rede que podem ser usados juntos ou separadamente:

- **Conectividade entre os recursos do Azure Stack**  
    Conecte recursos do Azure juntos em uma rede virtual segura e privada na nuvem.
- **Conectividade com a Internet**  
    Se comunicar de e para recursos do Azure Stack pela Internet.
- **Conectividade local**  
    Conecte uma rede local aos recursos do Azure Stack por meio de uma rede virtual privada (VPN) pela Internet ou por meio de uma conexão dedicada para o Azure Stack.
- **Direção do tráfego e balanceamento de carga**  
    Balancear o tráfego de servidores no mesmo local e direcione o tráfego para servidores em locais diferentes.
- **Segurança**  
    Filtre o tráfego de rede entre sub-redes ou máquinas virtuais (VM).
- **Roteamento**  
    Use o roteamento padrão ou controle totalmente o roteamento entre o Azure Stack e recursos locais.
- **Capacidade de gerenciamento**  
    Monitorar e gerenciar seus recursos de rede do Azure Stack.
- **Ferramentas de implantação e configuração**  
    Usar um portal baseado na web ou ferramentas de linha de comando de plataforma cruzada para implantar e configurar os recursos de rede.
