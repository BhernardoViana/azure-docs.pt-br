# <a name="introduction-to-azure-stack-networking"></a>Introdução à rede do Azure Stack

*Aplica-se a: Azure pilha integrado sistemas e o Kit de desenvolvimento de pilha do Azure*

A pilha do Azure fornece uma variedade de recursos de rede que podem ser usados juntos ou separadamente:

- **Conectividade entre os recursos da pilha do Azure**  
    Conecte-se a recursos do Azure juntos em uma rede virtual privada, seguro na nuvem.
- **Conectividade com a Internet**  
    Se comunicam em recursos da pilha do Azure pela Internet.
- **Conectividade local**  
    Conecte uma rede local aos recursos da pilha do Azure por meio de uma rede virtual privada (VPN) pela Internet ou por meio de uma conexão dedicada para a pilha do Azure.
- **Direção do tráfego e balanceamento de carga**  
    Balancear o tráfego para servidores no mesmo local e direcione o tráfego para servidores em diferentes locais.
- **Segurança**  
    Filtre o tráfego de rede entre sub-redes de rede ou máquinas virtuais (VM).
- **Roteamento**  
    Use o roteamento padrão ou controlar totalmente roteamento entre a pilha do Azure e recursos locais.
- **Capacidade de gerenciamento**  
    Monitorar e gerenciar sua pilha Azure recursos de rede.
- **Ferramentas de implantação e configuração**  
    Use um portal baseado na web ou ferramentas de linha de comando de plataforma cruzada para implantar e configurar os recursos de rede.
