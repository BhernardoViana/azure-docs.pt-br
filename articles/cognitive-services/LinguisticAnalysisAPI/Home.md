# <a name="what-is-the-linguistic-analysis-api-preview"></a>O que é a API de Análise Linguística? (Visualização)

> [!IMPORTANT]
> A versão prévia da Análise Linguística foi encerrada em 9 de agosto de 2018. Recomendamos usar os [módulos de Análise de Texto do Azure Machine Learning](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics) para análise e processamento de texto.

A API de Análise Linguística fornece acesso a ferramentas de NLP (processamento de linguagem natural) que identificam a estrutura do texto. A versão atual oferece três tipos de análise:

- [Separação de frases e geração de tokens](Sentences-and-Tokens.md)
- [Marcação de partes de fala](POS-tagging.md)
- [Análise de clientes](Constituency-Parsing.md)

## <a name="getting-started"></a>Introdução

Depois que estiver familiarizado com os principais tipos de análise listados acima, você poderá examinar o ponto de extremidade para [obter a lista de analisadores disponíveis](AnalyzersMethod.md) e como eles são [nomeados](Analyzer-Names.md) e, em seguida, o ponto de extremidade para [analisar a nova entrada](AnalyzeMethod.md).
