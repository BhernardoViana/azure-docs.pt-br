# <a name="recommendations-api-discontinued"></a>API de Recomendações (descontinuada)

A versão prévia da API de Recomendações não está mais em desenvolvimento ativo e foi oficialmente desativada em 15 de fevereiro. 

Para obter uma alternativa, considere o novo código-fonte aberto [modelo de Solução de Recomendações](https://aka.ms/recopcs), que permite hospedar um mecanismo de recomendações em sua própria assinatura do Azure. Ele oferece suporte para os cenários principais da API de Recomendações, enquanto também fornece a você a liberdade para implantar em qualquer região do Azure e controlar mais de perto os recursos computacionais usados para treinar modelos e atender as recomendações. 

Saiba mais sobre o modelo de Solução de Recomendações lendo [a documentação](https://github.com/Microsoft/Product-Recommendations).
