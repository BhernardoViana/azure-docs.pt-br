Este guia mostra como instalar o [SDK de Fala](~/articles/cognitive-services/speech-service/speech-sdk.md) para Java no Android. A configuração é baseada no pacote do SDK de Fala do Maven e do Android Studio 3.3.

> [!NOTE]
> Para o SDK dos Dispositivos de Fala e o dispositivo Roobo, confira [SDK dos Dispositivos de Fala](~/articles/cognitive-services/speech-service/speech-devices-sdk.md).

## <a name="supported-platforms"></a>Plataformas com suporte

O SDK de Fala é compatível com dispositivos Android que têm processadores ARM de 32/64 bits e Intel x86/x64 compatíveis.
