# <a name="get-the-cognitive-services-speech-devices-sdk"></a>Obter o SDK de Dispositivos de Fala dos Serviços Cognitivos

## <a name="requesting-access"></a>Solicitar acesso

O SDK de Dispositivos de Fala está em versão prévia restrita e exige seu registro no programa. Atualmente, a Microsoft prefere grandes empresas como candidatas para o acesso a esse produto.

Para obter acesso ao SDK de Dispositivos de Fala, execute estas etapas:

1. Vá para o [formulário de inscrição](https://aka.ms/sdsdk-signup) do SDK de Dispositivos de Fala da Microsoft.
1. Leia o [contrato de licença](speech-devices-sdk-license.md).
1. Se você concordar com os termos desse contrato de licença, selecione "Concordo".
1. Responda às perguntas no formulário.
1. Envie o formulário. 
1. Se o seu endereço de email ainda não fizer parte do Azure Active Directory, você receberá um email de convite, como este mostrado abaixo, após a aprovação. Se o seu endereço de email já estiver no Azure Active Directory, você receberá uma mensagem de email da equipe de Fala da Microsoft após a aprovação, e poderá pular para [Fazer download do SDK de Dispositivos de Fala](#download-the-speech-devices-sdk).

## <a name="approval-e-mail"></a>Email de aprovação

```
From: Microsoft Speech Team from Microsoft (via Microsoft) <invites@microsoft.com> 
Subject: You're invited to the Microsoft organization 
```

![mensagem de email](media/speech-devices-sdk/get-sdk-1.png)

## <a name="accept-access"></a>Aceitar o acesso
Execute as etapas abaixo para unir o Azure Active Directory com o endereço de email fornecido durante o registro. Esse processo concede a você acesso ao [site de download](https://shares.datatransfer.microsoft.com/) do SDK de Dispositivos de Fala.

1. Clique em **Introdução** na mensagem de email que você recebeu. Se sua organização já for cliente do Office 365, você receberá uma solicitação para entrar e poderá pular para a etapa 8.

2. Clique em **Avançar** na janela do navegador.

    ![janela de autenticação](media/speech-devices-sdk/get-sdk-2.png)

3. Crie uma conta Microsoft, caso ainda não tenha uma. Insira o mesmo endereço de email em que você recebeu o email de convite na etapa 6 acima.

    ![criar conta Microsoft](media/speech-devices-sdk/get-sdk-3.png)

4. Clique em **Avançar** para criar uma senha.

5. Quando receber uma solicitação para confirmar seu email, volte para a caixa de entrada para obter o código de verificação enviado a você.
 
7. Cole ou digite o código de segurança da mensagem de email na caixa de diálogo. Neste exemplo, é "8406". Em seguida, clique em **Próximo**.

    ![verificar email](media/speech-devices-sdk/get-sdk-6.png)
 
8. Quando você vir o Aplicativo do Painel de Acesso na janela do navegador, terá confirmado que seu endereço de email (da etapa 6) agora faz parte do Azure Active Directory. Agora você tem acesso ao site de download do SDK de Dispositivos de Fala.

## <a name="download-the-speech-devices-sdk"></a>Fazer download do SDK de Dispositivos de Fala

Acesse o [site de download do SDK de Dispositivos de Fala](https://shares.datatransfer.microsoft.com/) e faça logon com a conta Microsoft que você criou anteriormente. Agora, execute estas etapas para baixar o SDK de Dispositivos de Fala, o código de exemplo associado e o material de referência.

![site de download do sdk](media/speech-devices-sdk/get-sdk-7.png)

1. Faça o download e instale a ferramenta Aspera Connect quando receber a solicitação para fazê-lo pelo navegador.

    ![fazer download do Aspera Connect](media/speech-devices-sdk/get-sdk-8.png)
 
1. Clique em **Sim** para alternar para o Aspera Connect.

    ![alternar para o Aspera Connect](media/speech-devices-sdk/get-sdk-9.png)
 
1. Clique em **Permitir** para confirmar o download de arquivos com o Aspera Connect.

    ![fazer download com o Aspera Connect](media/speech-devices-sdk/get-sdk-10.png)
 
1. Feche a janela de Transferências do Aspera Connect após o download dos arquivos.

    ![Janela de Transferências do Aspera Connect](media/speech-devices-sdk/get-sdk-11.png)
 
Por padrão, os arquivos são baixados em sua pasta **Downloads**. Agora, você pode sair deste site.
