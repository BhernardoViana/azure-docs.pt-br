# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Como entrar na API de Tradução de Fala da Microsoft

Siga essas etapas para entrar na API de Tradução de Fala da Microsoft.

## <a name="1-sign-into-azure"></a>1: Entrar no Azure   
- Não tem uma conta? [Inscrever-se em uma conta da Microsoft Azure](http://azure.com/)
- Já tem uma conta? [Entrar](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: Conectar-se ao Microsoft Translator     
- Depois de entrar no Azure, vá para a seção [Serviços Cognitivos](https://portal.azure.com/#create/Microsoft.CognitiveServices)
- Em "Tipo de API", selecione a API da Tradução de Fala
- Na seção Tipo de preço, selecione o tipo de preço que melhor atende às suas necessidades
    - Cada assinatura tem um tipo gratuito. O tipo gratuito tem os mesmos recursos e funcionalidades que os planos pagos e não tem uma data de expiração.
    - É possível ter apenas uma assinatura gratuita para sua conta
- Preencha o restante do formulário e selecione o botão Criar
- A assinatura está pronta para uso imediato

## <a name="3-authentication-key"></a>3: Chave de autenticação    
- Recuperar sua chave de autenticação indo até o menu Todos os recursos > clique em sua assinatura
- O valor de Chave é usado para autenticação, mantenha o valor de chave confidencial. Você precisa do valor de chave ao desenvolver o seu aplicativo
