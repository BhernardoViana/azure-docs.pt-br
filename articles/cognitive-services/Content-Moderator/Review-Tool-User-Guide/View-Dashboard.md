# <a name="view-dashboard"></a>Exibir painel

Clique na guia **painel** para ver as métricas-chave para todos as revisões de conteúdo realizadas dentro da ferramenta. Você pode ver análises totais, concluídas e em andamento para imagens e texto. Você pode ver as métricas para sua equipe padrão em suas subequipes.
