# <a name="bing-visual-search-sdk"></a>SDK de Pesquisa Visual do Bing

A Pesquisa Visual do Bing fornece a seguinte SDKs cliente que você pode usar para exercer a API de Pesquisa Visual.

* [Amostras do .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Pacote NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [Bibliotecas .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) para definições e dependências
* [Exemplos do Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Bibliotecas Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) para definições e dependências
* [Amostras de Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Bibliotecas Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) para definições e dependências
* [Exemplos em Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Bibliotecas Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) para definições e dependências

Amostras do SDK para cada idioma incluem um arquivo Leiame com pré-requisitos para usar o SDK junto com as etapas para instalar e executar as amostras.
