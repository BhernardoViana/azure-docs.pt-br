# <a name="project-url-preview-use-and-display-requirements"></a>Requisitos de exibição e uso da Visualização de URL de Projeto

Use somente os dados de Visualização de URL de Projeto para visualizar snippets de código e imagens em miniatura com hiperlink para os sites de origem, no compartilhamento de URL iniciada pelo usuário final em mídias sociais, bot de bate-papo ou ofertas semelhantes. Você não deve copiar, armazenar ou armazenar em cache os dados recebidos da Visualização de URL de Projeto. Você deve cumprir todas as solicitações para desabilitar visualizações que você pode receber dos proprietários do site ou de conteúdo.
