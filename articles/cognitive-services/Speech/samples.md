# <a name="bing-speech-client-samples"></a>Exemplos de cliente de Fala do Bing

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-bing-speech-api-deprecation-note.md)]
 
A Fala do Bing fornece exemplos de ponta a ponta para demonstrar os diferentes casos de uso, por exemplo, reconhecimento de comando, reconhecimento contínuo e detecção de intenção. Todas as amostras estão disponíveis no GitHub e podem ser baixadas pelos seguintes links: O README.md em cada repositório, bem como a página [bibliotecas de clientes](GetStarted/GetStartedClientLibraries.md), fornecem detalhes sobre como compilar e executar as amostras.

- [Exemplos de API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Exemplos de JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Exemplos de cliente de área de trabalho C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Exemplos de serviço C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Java nas amostas de Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [ObjectiveC em amostras de iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Todos os SDKs dos Serviços Cognitivos da Microsoft e exemplos são licenciados com a licença do MIT. Para obter mais informações, consulte [LICENÇA](https://github.com/Azure-Samples/SpeechToText-REST/blob/master/LICENSE).
