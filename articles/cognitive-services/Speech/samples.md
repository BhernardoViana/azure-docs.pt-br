# <a name="microsoft-speech-client-samples"></a>Amostras de cliente de fala da Microsoft

O Serviço de Fala da Microsoft fornece amostras de ponta a ponta mostrando como usar a API de reconhecimento de fala da Microsoft em casos de uso diferentes, por exemplo o reconhecimento de comando, reconhecimento contínuo e detecção de intenção. Todos os exemplos estão disponíveis no GitHub e podem ser baixados pelos links a seguir: o README.md em cada repositório, bem como a [página de bibliotecas de cliente](GetStarted/GetStartedClientLibraries.md) fornecem detalhes sobre como criar e executar os exemplos.

- [Exemplos de API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Exemplos de JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Exemplos de cliente de área de trabalho C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Exemplos de serviço C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Java nas amostas de Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [ObjectiveC em amostras de iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Todos os SDKs dos Serviços Cognitivos da Microsoft e exemplos são licenciados com a licença do MIT. Para obter mais informações, consulte [LICENÇA](https://github.com/Microsoft/Cognitive-Speech-STT-JavaScript/blob/master/LICENSE.md).
