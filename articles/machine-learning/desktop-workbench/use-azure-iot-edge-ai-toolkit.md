# <a name="use-the-ai-toolkit-for-azure-iot-edge"></a>Usar o Kit de ferramentas de IA para o Azure IoT Edge

[!INCLUDE [workbench-deprecated](../../../includes/aml-deprecating-preview-2017.md)] 

Leve o poder da inteligência artificial, do aprendizado de máquina e da análise avançada até o limite. Com o Kit de ferramentas de IA para o Azure IoT Edge, você pode implantar a IA e o aprendizado de máquina em que residem os dados. Operacionalize seu modelo como um contêiner do Docker com uma API REST e, em seguida, envie o modelo por push para um dispositivo de borda com o Hub IoT do Azure. O Kit de ferramentas de IA para o Azure IoT Edge leva inteligência, computação elástica e o poder da nuvem para lugares com conectividade limitada, intermitente ou inexistente.

O Kit de ferramentas de IA para o Azure IoT Edge é uma coleção de scripts, códigos e contêineres implantáveis. Exemplos incluem manutenção preditiva, classificação de imagens e processamento de fala, bem como a implantação de modelo personalizado por meio do Azure Machine Learning e do Hub IoT do Azure. Os modelos incluídos com o kit de ferramentas podem ser usados no estado em que se encontram, mas todo o código-fonte e os dados estão disponíveis para os desenvolvedores personalizarem de acordo com suas necessidades.

O repositório GitHub público do Kit de ferramentas de IA para o Azure IoT Edge pode ser encontrado em [aka.ms/AI-toolkit](https://aka.ms/AI-toolkit).
