# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>Preparar os dados com o SDK de preparação de dados do Azure Machine Learning para Python
 
A preparação de dados é uma parte importante de um fluxo de trabalho de aprendizado de máquina. Seus modelos poderão ser mais precisos e eficientes se tiverem acesso para limpar dados em um formato que é mais fácil de consumir. Você pode usar o [SDK de preparação de dados do Azure Machine Learning](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) para carregar dados de vários formatos, transformá-los para serem mais utilizáveis e gravá-los em um local para serem acessados por seus modelos.
