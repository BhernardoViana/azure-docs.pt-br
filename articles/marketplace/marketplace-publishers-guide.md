# <a name="azure-marketplace-and-appsource-publisher-guide"></a>Guia do Editor do Azure Marketplace e do AppSource  

## <a name="overview"></a>Visão geral 

Bem-vindo ao guia do editor do Marketplace. Elaboramos este guia para ajudar os editores, tanto iniciantes como experientes, a aproveitar as vitrines do Microsoft Azure Marketplace (www.azuremarketplace.com) e do Microsoft AppSource (www.appsource.com) para ampliar os negócios em parceria com a Microsoft, listando os aplicativos e serviços.  

Este artigo aborda os seguintes tópicos: 
*   O que é o Marketplace e como ele funciona 
*   Diferenças entre as vitrines do Microsoft Azure Marketplace e do Microsoft AppSource 
*   Vantagens da participação no Marketplace 
*   Como se tornar editor e listar um aplicativo ou serviço 
*   Como ampliar os negócios 

Este guia contém informações técnicas e comerciais sobre os procedimentos que você deve entender antes de listar os aplicativos e serviços. Use a barra de navegação localizada à esquerda desta página para ir até outras seções deste guia, e navegue diretamente para os tópicos principais.   

Se tiver dúvidas sobre o Microsoft Azure Marketplace e o Microsoft AppSource, envie-nos uma mensagem em Tipo de problema > Integração do Marketplace. Nossa equipe de integração de editores do Marketplace terá satisfação em ajudá-lo! 

---  

## <a name="benefits-of-participating-in-the-marketplace"></a>Benefícios da participação no marketplace 

O Marketplace é o ponto de partida das atividades de entrada no mercado conjuntas com a Microsoft para impulsionar o crescimento dos negócios. Usando a promoção de lançamento, a geração de demanda e as iniciativas conjuntas de marketing e vendas, as ofertas do marketplace podem ser o ponto central do mecanismo de negócios na nuvem. Não há taxas para participação no marketplace.  

Nosso objetivo é conectar clientes da Microsoft com as melhores soluções oferecidas pelo nosso ecossistema parceiro. Para fazer isso, nós o auxiliamos em toda sua jornada, que envolve a integração, a publicação e o crescimento. 

Aproveite as funcionalidades do marketplace para ampliar os negócios: 

### <a name="expand-to-new-markets-and-segments-and-generate-new-sales-opportunities"></a>Desenvolva novos mercados e segmentos, e crie oportunidades de vendas.

Use o marketplace como um canal para acessar novos mercados, segmentos e usuários de nuvem da Microsoft com um portfólio de soluções. Desenvolva novos clientes potenciais e cultive-os para gerar oportunidades de vendas. Aproveite as atividades conjuntas de marketing e vendas para adquirir mais clientes.  

### <a name="enhance-business-value-and-increase-deal-size-with-existing-and-new-customers"></a>Melhore o valor comercial e aumente o tamanho do negócio com clientes novos e existentes 

Comercialize as soluções com vendas adicionais e cruzadas para atender às necessidades dos clientes, depois de migrar as cargas de trabalho para a nuvem. Se vende ou implanta por meio do marketplace, você pode reduzir os ciclos de vendas, agilizar os projetos e aumentar a lucratividade dos negócios vendendo soluções completas direcionadas para cargas de trabalho e cenários específicos do setor. 
  
### <a name="get-actionable-insights"></a>Obtenha insights acionáveis 

Obtenha informações sobre o desempenho das listagens, por meio das análises que disponibilizamos para você. Saiba mais sobre desempenho, pedidos e pagamentos, bem como maximizar as atividades de campanha da solução.  

### <a name="marketplace-go-to-market-benefits"></a>Vantagens dos Serviços Microsoft Go-To-Market 

As novas listagens do marketplace são qualificadas para um conjunto diversificado de benefícios gratuitos que podem ajudar os parceiros a desenvolver negócios. Esses benefícios se enquadram nas seguintes categorias: 

*   Técnica: tudo o que é necessário para preparar o aplicativo para lançamento. Envolve suporte técnico, criação do aplicativo, design de arquitetura e créditos Azure para desenvolvimento e teste. 

*   Ações conjuntas de marketing: tudo o que é necessário para lançar a oferta. Acesse conceitos gratuitos dos Serviços Microsoft Go-To-Market, que podem ajudá-lo a lançar e a promover soluções. Você pode também se qualificar para a inclusão de uma campanha de marketing adicional da Microsoft e criar oportunidades para se destacar no marketplace.  

*   Vendas conjuntas: comece ou impulsione sua jornada para vender com a Microsoft no marketplace. Acesse programas e suporte para gerar vendas conjuntas com as equipes da Microsoft.  

Quando [enviar a solução](https://azuremarketplace.microsoft.com/sell/signup) ao marketplace, nossa equipe de integração entrará em contato com você para auxiliá-lo na experiência de publicação.  

Para saber mais sobre as vantagens dos Serviços Microsoft Go-To-Market e ver maneiras de ampliar os negócios no marketplace, confira a [página Serviços Microsoft Go-To-Market](https://partner.microsoft.com/reach-customers/gtm).

## <a name="marketplace-and-storefronts"></a>Vitrines do Marketplace 

A Microsoft fornece duas vitrines diferentes no marketplace: o AppSource e o Azure Marketplace. As vitrines do marketplace estão alinhadas com o público e os produtos de nuvem da Microsoft para ajudar os clientes a localizar exatamente o que precisam. 

Escolher uma vitrine começa pela identificação do público-alvo da solução. Use a tabela a seguir para decidir: 

|  | AppSource | Azure Marketplace |
|:--- |:--- |:--- |
| Público-alvo | Tomadores de decisão da linha de negócios | Profissionais e desenvolvedores de TI | 
| Criado com ou para estender | Azure<br />Dynamics 365<br />Office 365<br />Power BI<br />PowerApps | Azure |  

Cada vitrine fornece opções e recursos especializados para maximizar as oportunidades de negócios. Para obter uma comparação mais detalhada das nossas vitrines, confira o artigo [Comparação entre o Azure Marketplace e o AppSource](./comparing-appsource-azure-marketplace.md).
