<a name="lcs-subscription"></a>Assinatura do LCS
================

Neste tutorial, você aprenderá como criar uma assinatura. Esta assinatura lhe dará um locatário de teste do Microsoft Online e um projeto do Microsoft Dynamics Lifecycle Services onde você pode implantar um ambiente. Você também definirá usuários adicionais em seu locatário do Microsoft Online e obterá experiência com os seguintes recursos de administração de serviço. Aqui estão as habilidades que você aprenderá:

- Assinar e criar um novo locatário de teste Microsoft Online.
- Navegar para projetos do Lifecycle Services.
- Usando recursos no Lifecycle Services.
- Adicionando usuários adicionais ao Microsoft Azure Active Directory e o cliente.
- Visualizar recursos no seu email de assinatura.

<a name="key-terms"></a>Principais termos
=========

**Locatário do Microsoft Online Services** - um locatário é o grupo de todas as assinaturas e os usuários para sua organização. O locatário é criado ao mesmo tempo de primeira assinatura do Microsoft Online Services.

**Assinatura** - uma assinatura proporciona um ambiente online. Esse ambiente permite testar o Dynamics 365 para operações na nuvem e também permite que você veja como as personalizações que você desenvolve podem ser implantadas para a nuvem.

**Microsoft Azure Active Directory** - o ambiente de nuvem inclui o Microsoft Azure Active Directory (AD), que ajuda você a gerenciar usuários, grupos, funções de segurança e licenças para aplicativos online, de maneira semelhante à forma como gerenciá-los para os ambientes locais.

**Usuários** - usuários dos serviços que a sua organização assinou são gerenciados no Microsoft Azure Active Directory. Todos os usuários em seu locatário podem ser adicionados e atribuídos a funções de segurança.

**Desenvolvedores e administradores** - desenvolvedores e administradores são os usuários que também têm acesso aos Lifecycle Services que permite que eles gerenciem projetos e ambientes. Esses usuários também são os usuários finais.

Veja [Assinatura LCS](https://docs.microsoft.com/dynamics365/operations/dev-itpro/dev-tools/sign-up-preview-subscription) para obter mais informações.
