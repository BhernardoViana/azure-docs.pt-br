# <a name="azure-marketplace-and-appsource-roadmap"></a>Roteiro do Azure Marketplace e do AppSource

Este documento fornece uma exibição sobre as novidades para o Azure Marketplace, AppSource e programas de parceiros relacionados. Ele captura alguns recursos importantes já confirmados e um período de tempo aproximado para quando você pode esperar vê-los. Não é uma lista completa de todos os novos recursos, mas serve para fornecer visibilidade sobre nossos principais investimentos. Esses conjuntos de recursos e os períodos de tempo de entrega são atuais e sujeitos a alterações. Esta página é atualizada mensalmente.

Suas sugestões são bem-vindas! Participe da conversa na [Comunidade de Parceiros da Microsoft](https://www.microsoftpartnercommunity.com/) para obter as atualizações mais recentes sobre os novos recursos, programas e eventos.
