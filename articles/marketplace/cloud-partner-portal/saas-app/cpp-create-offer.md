# <a name="create-a-new-saas-application-offer"></a>Criar uma nova oferta de aplicativo SaaS

Este artigo descreve como criar e publicar uma entrada de oferta de aplicativo SaaS para o Azure Marketplace.

> [!IMPORTANT] 
> SaaS oferecem funcionalidade está sendo migrada para o [Microsoft Partner Center](https://partner.microsoft.com/dashboard/directory).  Todos os novos editores devem usar o Partner Center para criar novas ofertas de SaaS e gerenciando ofertas existentes.  Editores atuais com ofertas de SaaS estão sendo migrados batchwise do Portal de parceiro do Cloud Partner Center.  Portal do Cloud Partner exibirá mensagens de status para indicar quando as ofertas específicas do existentes foram migradas.


## <a name="offer-process"></a>Processo de oferta

O diagrama a seguir mostra o processo de criação de uma oferta de aplicativo SaaS.

![Processo para criar uma oferta de SaaS](./media/saas-offer-process-overview.png)

## <a name="offer-components"></a>Componentes de oferta

A oferta de aplicativo SaaS consiste em cinco seções, descritas na tabela a seguir:

|  **Grupo de ativos**   |  **Descrição**  |
|  ---------------   |  ---------------  |
|    Configurações da oferta  |  Use para configurar uma identidade exclusiva para o aplicativo SaaS.                 |
|  Informações técnicas    |  Use para configurar o tipo de solução SaaS e fornecer os detalhes de conexão para seu aplicativo.                |
|  Informações do Canal      |   Fornece informações de canal, como materiais GTM e contatos.                |
|  Test Drive        |   Seção opcional para definir um serviço que permitirá que os clientes testem a oferta antes de comprá-la.                |
|  Detalhes da vitrine       | Contém os ativos de marketing, jurídicos e de cliente potencial e especificações.  <ul><li> Ativos de marketing incluem nome da oferta, descrição e logotipos</li> <li> Ativos jurídicos incluem uma política de privacidade, termos de uso e outras documentações legais</li>  <li> A política de gerenciamento de cliente potencial permite que você especifique como manusear os clientes potenciais do portal do usuário final do Microsoft Azure Marketplace.</li> </ul> |
| Contatos            | Contém informações de contato e política de suporte |

## <a name="new-offer-form"></a>Novo formulário de oferta

Entre no [Portal do Cloud Partner](https://cloudpartner.azure.com/)e, em seguida, selecione **+ Nova oferta** na barra de menus à esquerda. No menu Nova oferta, selecione **Aplicativos SaaS** para exibir o formulário Nova oferta e iniciar o processo de definição de ativos para uma nova oferta de aplicativo SaaS.

![Menu Nova oferta para aplicativos SaaS](./media/azure-new-saas-offer.png)
