# <a name="azure-saas-application-offer"></a>Oferta de aplicativo de SaaS do Azure

O [Portal do Cloud Partner](https://cloudpartner.azure.com/) não oferece suporte a criação e gerenciamento de ofertas de aplicativos SaaS.  Em vez disso, essa funcionalidade foi movida para o Microsoft [portal do Partner Center](https://partner.microsoft.com/pcv/).  Para obter mais informações, consulte [criar uma nova oferta de SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Próximas etapas

Antes de criar uma nova oferta de aplicativo de SaaS, você deve ler o [guia de publicação da oferta de aplicativos de SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) e examine as etapas de portal do Partner Center listadas na [lista de verificação de criação de oferta](../../partner-center-portal/offer-creation-checklist.md).
