# <a name="create-power-bi-app-technical-assets"></a>Criar ativos técnicos do Aplicativo do Power BI

O principal ativo técnico necessário para esse tipo de oferta é um Aplicativo do Power BI, que é uma coleção de um conjunto de dados primário, relatório e (ou) painel, além dos serviços conectados e conjuntos de dados inseridos opcionais. (Os Aplicativos do Power BI são uma evolução do tipo de ativo anteriormente conhecido como [pacotes de conteúdo](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction).)  Para obter informações completas sobre como desenvolver esses aplicativos, confira [Aplicativos do Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtaining-an-installation-url"></a>Obter uma URL de instalação

A compilação do Aplicativo do Power BI só está disponível no [serviço do Power BI](https://powerbi.microsoft.com/) e requer logon usando uma [licença do Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Depois que seu aplicativo de serviço é criado e testado no Power BI, salve a URL de instalação do aplicativo gerada pelo Power BI.  Você deve fornecer essa URL na [guia Informações Técnicas](./cpp-technical-info-tab.md) do formulário de envio do Portal do Cloud Partner.
 

## <a name="next-steps"></a>Próximas etapas

Depois de criar e testar o Aplicativo do Power BI e criar a oferta associada, você poderá [publicar a oferta do Power BI](./cpp-publish-offer.md).
