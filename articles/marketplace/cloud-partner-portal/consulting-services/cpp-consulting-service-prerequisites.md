# <a name="consulting-service-prerequisites"></a>Pré-requisitos do serviço de consultoria

Para enviar ofertas de serviço de consultoria para qualquer um dos produtos abaixo, você precisa primeiro atender a um ou mais dos requisitos correspondentes.

* **Dynamics 365 for Customer Engagement**: Os parceiros devem ter uma certificação Prata ou Ouro para a [competência Cloud Customer Relationship Management](https://partner.microsoft.com/membership/cloud-customer-relationship-management-competency). 
* **Dynamics 365 for Finance & Operations**: Os parceiros devem ter uma certificação Prata ou Ouro para a [competência Enterprise Resource Planning](https://partner.microsoft.com/membership/enterprise-resource-planning-competency) e uma receita mínima de US$ 25 mil em operações em nuvem nos últimos 12 meses.
* **Dynamics 365 Business Central**: os parceiros devem ter agido como CSP (Provedor de Solução de Nuvem) ou DPOR (Parceiro de Registro Digital) para pelo menos um cliente.
    * **CSP**: veja [Introdução ao CSP](https://partner.microsoft.com/cloud-solution-provider/get-started) para saber como se tornar um CSP.
    * **DPOR**: veja [Visão Geral de DPOR](https://partner.microsoft.com/membership/digital-partner-of-record#Membership_Digital_Partner_of_Record_Navigated_Rich_Text_Node1) para requisitos de DPOR.
* **Power BI**: seja listado na demonstração do parceiro do Power BI. veja [Critérios do Parceiro de Solução](https://powerbi.microsoft.com/become-a-partner) para obter mais informações.
* **PowerApps**: Os parceiros devem ter uma solução publicada na [Demonstração de Parceiro](https://powerapps.microsoft.com/partner-showcase).

**Próximas etapas**

Se os requisitos acima forem atendidos, você estará pronto para [criar uma oferta de serviços de consultoria](./cpp-consulting-service-create-offer.md).
