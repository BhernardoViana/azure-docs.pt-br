# <a name="consulting-service-prerequisites"></a>Pré-requisitos do serviço de consultoria

Para enviar ofertas de serviço de consultoria para qualquer um dos produtos abaixo, você precisa atender a um ou mais dos requisitos correspondentes:

* **Dynamics 365 for Customer Engagement**: Os parceiros devem ter uma certificação Prata ou Ouro para a [competência Cloud Customer Relationship Management](https://partner.microsoft.com/membership/cloud-customer-relationship-management-competency). 
* **Dynamics 365 for Finance & Operations**: Os parceiros devem ter uma certificação Prata ou Ouro para a [competência de Planejamento de Recursos Empresariais](https://partner.microsoft.com/membership/enterprise-resource-planning-competency). Os parceiros também devem ter uma receita mínima de US$ 25 mil em Operações em Nuvem nos 12 meses finais.
* **Dynamics 365 Business Central**: os parceiros devem ter agido como CSP (Provedor de Solução de Nuvem) ou DPOR (Parceiro de Registro Digital) para pelo menos um cliente.
    * **CSP**: Para obter informações sobre como se tornar um CSP, confira [Começar como um CSP](https://partner.microsoft.com/cloud-solution-provider/get-started).
    * **DPOR**: Para requisitos de DPOR, confira a [Visão geral de DPOR](https://partner.microsoft.com/membership/digital-partner-of-record#Membership_Digital_Partner_of_Record_Navigated_Rich_Text_Node1).
* **Power BI**: seja listado na demonstração do parceiro do Power BI. Para mais informações, confira [Critérios do Parceiro de Solução](https://powerbi.microsoft.com/become-a-partner).
* **PowerApps**: Os parceiros devem ter uma solução publicada no [Partner Solution Showcase](https://powerapps.microsoft.com/partner-showcase).

## <a name="next-steps"></a>Próximas etapas
