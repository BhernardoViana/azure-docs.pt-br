# <a name="managing-users-on-the-cloud-partner-portal"></a>Gerenciando usuários no Portal do Cloud Partner

O [Portal do Cloud Partner](https://cloudpartner.azure.com) permite que você adicione os usuários com acesso baseado em função para ofertas de máquina virtual. Isso ajuda a gerenciar o acesso e as permissões enquanto várias pessoas estão trabalhando em ofertas.

## <a name="add-users-and-assign-roles"></a>Adicionar usuários e atribuir funções 

Use as seguintes etapas para adicionar usuários e atribuir funções para publicação de oferta.

1. Clique o **usuários** guia no painel de navegação do lado esquerdo.

    ![Guia Usuários](./media/userstab.png)


2. Clique em **Adicionar Usuário**.

    ![Selecione Adicionar usuário](./media/adduser.png)


3. Digite o endereço de email do usuário e selecione uma atribuição de função.  Você pode adicionar o novo usuário como "proprietário" ou "Colaborador".

    **Email**: adicione os endereços de email dos colegas de equipe que trabalharão na publicação da oferta. Há suporte para contas da Microsoft (Outlook, Hotmail e Live) e as IDs da organização.

    - Adicionar um "grupo do team email alias/grupo de segurança" para se proteger contra a pessoa que sair da organização.
    - Certifique-se de que email IDs fornecidas no Portal de parceiros de nuvem são monitoradas para qualquer comunicação da Microsoft.
    
    ![Atribuir função](./media/assignrole.png)

    **Função**: confira a tabela abaixo para identificar o tipo de função de usuário apropriada.

    ![Nível de funções](./media/roleaccesslevel.png)

    Somente os proprietários podem acessar pagamentos do Azure e clientes Azure guias [Seller Insights](../../cloud-partner-portal-orig/si-getting-started.md).


4. Selecione o nome do editor que você deseja adicionar ao novo usuário e, em seguida, clique em Adicionar para concluir a adição do usuário.

    > [!NOTE]
    > Os "nomes de publicador" para o qual seu email é adicionado como um proprietário ou colaborador serão exibidos na lista suspensa de **publicador** para seleção.


    ![Atribuir função - selecione publicador](./media/assignselectpublisher.png)


## <a name="delete-an-existing-user"></a>Excluir um usuário existente

Para remover um usuário existente, pesquise seus emails na **os usuários** guia e, em seguida, pressione a **excluir** botão no lado direito.
