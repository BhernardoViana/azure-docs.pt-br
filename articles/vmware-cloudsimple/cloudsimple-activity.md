# <a name="activity-management-overview"></a>Visão geral do gerenciamento de atividade

CloudSimple rastreia todas as atividades que podem afetar o funcionamento de suas nuvens privadas. Essas atividades, incluindo alertas, eventos, tarefas e atividade de auditoria. O [páginas de atividade](https://docs.azure.cloudsimple.com/activity/) resumir a atividade atual e permitem que você fazer uma busca detalhada para obter mais detalhes.

## <a name="events"></a>Events

Eventos de acompanhar a atividade do sistema e usuário no portal CloudSimple.  Eventos mostram a atividade associada a um recurso específico e a gravidade do impacto.  Você pode exibir os eventos a partir do portal CloudSimple.

## <a name="alerts"></a>Alertas

Alertas são notificações de qualquer atividade significativa em seu ambiente CloudSimple.  Eventos que afetam o acesso de usuário ou de cobrança, são mostrados como alertas.  Você pode reconhecer os alertas gerados a partir do portal CloudSimple.

## <a name="tasks"></a>Tarefas

Tarefas de acompanhar qualquer operação de usuário, o que leva mais de 30 segundos para ser concluída.  Você pode monitorar o progresso de uma tarefa a partir do portal CloudSimple.  Concluída a mostrar as tarefas de tempo total decorrido para a tarefa seja concluída.

## <a name="audit"></a>Audit

Operações de acompanhar o usuário realizou logs de auditoria.  Os logs contêm os parâmetros que o usuário fornecido para a operação.  Você pode usar os logs de auditoria para monitorar a atividade de usuário para todos os usuários.
