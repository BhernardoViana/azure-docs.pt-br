# <a name="iot-solution-accelerators-documentation"></a>Documentação dos aceleradores de soluções de IoT

Comece a usar rapidamente os aceleradores de solução de IoT para criar soluções de IoT. Inclua novos dispositivos e conecte os existentes usando SDKs de dispositivos para diversas plataformas, incluindo Linux, Windows e sistemas operacionais em tempo real. Dimensione com facilidade apenas alguns sensores até milhões de dispositivos conectados simultaneamente e conte com a disponibilidade global do Azure&mdash;independentemente do tamanho do seu projeto.
