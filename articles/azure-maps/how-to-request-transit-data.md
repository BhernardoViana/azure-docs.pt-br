# <a name="request-public-transit-data-using-the-azure-maps-mobility-service"></a>Solicitar dados de trânsito público usando o serviço de mobilidade do Azure mapas 

Este artigo mostra como usar o Azure mapas [o serviço de mobilidade](https://aka.ms/AzureMapsMobilityService) solicitar trânsito público dados, incluindo paradas, informações de rota em estimativas de tempo de viagem.

Neste artigo você aprenderá como:

* Obter uma ID de área metropolitana usando o [obter API do Metro área](https://aka.ms/AzureMapsMobilityMetro)
* Solicitar trânsito próximo deixará de usar [obter próximos trânsito](https://aka.ms/AzureMapsMobilityNearbyTransit) service.
* Consulta [API de rotas de trânsito obter](https://aka.ms/AzureMapsMobilityTransitRoute) para planejar uma rota por meio do trânsito público.
* Solicitar a geometria de rotas de trânsito e agenda detalhada para a rota usando o [obter API do itinerário trânsito](https://aka.ms/ https://azure.microsoft.com/services/azure-maps/).


## <a name="prerequisites"></a>Pré-requisitos

Para fazer todas as chamadas para o trânsito de mapas do Azure pública APIs, você precisa de uma conta de mapas e uma chave. Para obter mais informações sobre como criar uma conta e recuperar uma chave, consulte [Como gerenciar as chaves e a conta dos Mapas do Azure](how-to-manage-account-keys.md).

Este artigo usa o [aplicativo Postman](https://www.getpostman.com/apps) para criar chamadas REST. Você pode usar qualquer ambiente de desenvolvimento de API que você preferir.


## <a name="get-a-metro-area-id"></a>Obter uma ID de área metropolitana

Para solicitar informações de trânsito para uma determinada área metropolitana, será necessário o `metroId` para a área que você deseja solicitar os dados em trânsito para. [Obter a API de área de Metro](https://aka.ms/AzureMapsMobilityMetro) permite que você solicite as áreas de metrô em que o serviço de mobilidade de mapas do Azure está disponível. Resposta incluir detalhes tais como `metroId`, `metroName` e uma representação da geometria área metropolitana no formato GeoJSON.

Vamos fazer uma solicitação para obter a área do Metro para ID de área metropolitana de Seattle Tacoma. Para a ID da solicitação para uma área metropolitana, conclua as seguintes etapas:

1. Crie uma coleção para armazenar as solicitações. No aplicativo Postman, selecione **New**. No **criar novo** janela, selecione **coleção**. Nome da coleção e selecione o **criar** botão.

2. Para criar a solicitação, selecione **New** novamente. No **criar novo** janela, selecione **solicitar**. Insira um **nome da solicitação** para a solicitação, selecione a coleção que você criou na etapa anterior como o local no qual salvar a solicitação e, em seguida, selecione **salvar**.

    ```HTTP
    https://atlas.microsoft.com/mobility/metroArea/id/json?subscription-key={subscription-key}&api-version=1.0&query=47.63096,-122.126

## <a name="request-nearby-transit-stops"></a>Solicitar nas proximidades de paradas de trânsito

Os mapas do Azure [obter próximos trânsito](https://aka.ms/AzureMapsMobilityNearbyTransit) serviço permite que você pesquise os objetos de trânsito, por exemplo, o trânsito público para e compartilhados de bicicletas em torno de um determinado local, retornando o trânsito detalhes do objeto. Em seguida, será fazer uma solicitação para o serviço para pesquisar próximo trânsito público para em 300 metros raio em torno de determinado local. Na solicitação, é preciso incluir o `metroId` recuperados anteriormente.

Para fazer uma solicitação para o [obter próximos trânsito](https://aka.ms/AzureMapsMobilityNearbyTransit), siga as etapas abaixo:

1. No Postman, clique em **nova solicitação** | **solicitação GET** e nomeie-a **interrompe obter próximos**.

2. Na guia criador, selecione a **Obtenha** método HTTP, insira a URL de solicitação a seguir para seu ponto de extremidade de API e clique em **enviar**.

    ```HTTP
    https://atlas.microsoft.com/mobility/transit/nearby/json?subscription-key={subscription-key}&api-version=1.0&metroId=522&query=47.63096,-122.126&radius=300&objectType=stop

Serviço fornece uma variedade de modos de viagem, incluindo andar, andar de bicicleta e trânsito público. Em seguida, irá procurar uma rota de parada mais próximo de barramento para o Space Needle em Seattle.

1. No Postman, clique em **nova solicitação** | **solicitação GET** e nomeie-a **informações de rota Get**.

2. Na guia criador, selecione a **Obtenha** método HTTP, insira a URL de solicitação a seguir para seu ponto de extremidade de API e clique em **enviar**.

## <a name="request-fastest-route-itinerary"></a>Roteiro de rota mais rápido de solicitação

Os mapas do Azure [obter itinerário de trânsito](https://aka.ms/AzureMapsMobilityTransitItinerary) serviço permite que você solicite dados de uma rota específica usando a rota **itinerário ID** retornado pela [obter API de rotas de trânsito](https://aka.ms/AzureMapsMobilityTransitRoute) serviço.
