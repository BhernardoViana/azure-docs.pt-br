# <a name="about-azure-networking-msp-partners"></a>Sobre os parceiros do MSP de rede do Azure

Os serviços de rede do permite que o programa de parceiro do Azure à rede gerenciada MSP (Service Provider) voltada para MSPs, empresas de telecomunicações e integradores de sistemas (SIs) para oferecer a nuvem e híbridas centralizados de serviços de rede em torno do portfólio do Azure dos produtos de rede e serviços.

MSPs na rede do Azure são um conjunto especializado de provedores de serviços gerenciados que resolvem os desafios e as necessidades de rede de nuvem empresarial em todos os aspectos da nuvem e uma rede híbrida. Os serviços de rede gerenciado incluam um ou mais dos seguintes serviços: arquitetura, planejamento, implantação, operações, manutenção e otimização de rede.

![MSP][0]

Os clientes podem adquirir ofertas de serviços gerenciados pelo MSP no Azure Marketplace, ou diretamente do MSP. MSPs podem implantar e gerenciar recursos de rede do Azure em assinaturas de clientes e grupos de recursos usando o farol do Azure.
