# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Notas de versão do Azure dados caixa Gateway 1905 e borda da caixa de dados do Azure

## <a name="overview"></a>Visão geral

As notas de versão a seguir identificam os problemas críticos abertos e os problemas resolvidos para a versão de 1905 para a borda de caixa de dados do Azure e o Gateway de caixa de dados do Azure.

As notas de versão são continuamente atualizadas e, à medida que são descobertas questões críticas que exijam uma solução alternativa, elas são adicionadas. Antes de implantar seu Gateway de caixa de dados de caixa de borda/dados, examine cuidadosamente as informações contidas nas notas de versão.

Esta versão corresponde às versões de software:

- **Gateway de caixa de dados 1905 (1.6.887.626)**
- **Borda da caixa de dados 1905 (1.6.887.626)**

> [!NOTE]
> Atualização de 1905 pode ser aplicada somente a dispositivos de borda da caixa de dados que estão executando a versão GA do software.

## <a name="whats-new"></a>Novidades

- **Aprimoramentos de log do campo programável Gate Array (FPGA)** - nesta versão, fizemos no registro em log e alerta aprimoramentos relacionados ao FPGA. Esta é uma atualização necessária para dados de caixa de borda, se você estiver usando o recurso de computação de borda com o FPGA. Para obter mais informações, consulte como [transformar dados com a computação de borda na borda da caixa de dados](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Problemas conhecidos na versão de GA

Nenhum novo problema é observados para esta versão. Todos os da versão observado problemas foram herdados das versões anteriores. Para ver uma lista dos problemas conhecidos, acesse [problemas conhecidos na versão GA](data-box-gateway-release-notes.md#known-issues-in-ga-release).
