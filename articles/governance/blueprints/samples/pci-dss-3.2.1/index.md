# <a name="overview-of-the-pci-dss-v321-blueprint-sample"></a>Visão geral da amostra do blueprint do PCI-DSS v3.2.1

A amostra do blueprint do PCI-DSS v3.2.1 é um conjunto de políticas que ajuda na conformidade com o PCI-DSS v3.2.1. Este blueprint ajuda os clientes a administrarem ambientes baseados em nuvem com cargas de trabalho de PCI-DSS.
O blueprint do PCI-DSS implanta um conjunto principal de políticas para arquiteturas implantadas pelo Azure que exigem esta certificação.

## <a name="control-mapping"></a>Mapeamento de controle

A seção de mapeamento de controle fornece detalhes sobre as políticas incluídas dentro desta iniciativa e como essas políticas ajudam a atender a vários controles definidos pelo PCI-DSS v3.2.1. Quando atribuídos a uma arquitetura, os recursos são avaliados pelo Azure Policy para verificar a não conformidade com políticas atribuídas.

Depois de atribuir esse blueprint, exiba o nível de conformidade de seus ambientes do Azure no Painel de Conformidade do Azure Policy.
