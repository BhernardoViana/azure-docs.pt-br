# <a name="check-usage-of-free-services-included-with-your-azure-free-account"></a>Verificar o uso de serviços gratuitos incluídos em sua conta gratuita do Azure

Você não é cobrado pelos serviços incluídos gratuitamente em sua conta gratuita do Azure, a menos que exceda os limites dos serviços. Para permanecer nos limites, use o portal do Azure para acompanhar o uso do serviço gratuito.

## <a name="check-usage-in-the-azure-portal"></a>Verificar uso no portal do Azure

1.  Entre no [Portal do Azure](https://portal.azure.com).

2.  Pesquise **Assinaturas**.

    ![Captura de tela que mostra a pesquisa no portal para assinaturas](./media/billing-check-usage-of-free-services/billing-search-subscriptions.png)

3.  Selecione a assinatura que foi criada quando você se inscreveu em sua conta gratuita do Azure.

4.  Role a página para baixo para encontrar a tabela que mostra o uso do serviço gratuito.

    ![Captura de tela que mostra o uso de serviços gratuitos](./media/billing-check-usage-of-free-services/subscription-usage-free-services.png)

    A tabela tem as seguintes colunas:

* **Medidor:** Identifica a unidade de medida para o serviço que está sendo consumido.
* **Uso/Limite:** uso e o limite para o medidor no mês atual.
* **Status:** Status de uso do serviço. Com base em seu uso, você pode ter um dos seguintes status:
  * **Não está em uso:** você não usou o medidor ou o uso desse medidor não alcançou o sistema de cobrança.
  * **Excedido em \<Data>:** você excedeu o limite do medidor em \<Data>.
  * **Provavelmente não excederá:** é improvável que você exceda o limite do medidor.
  * **Excede em \<Data>:** é improvável que você exceda o limite do medidor em \<Data>.

> [!IMPORTANT]
>
> Os serviços gratuitos só estão disponíveis para a assinatura que foi criada quando você se inscreveu em sua conta gratuita do Azure. Caso você não consiga ver a tabela de serviços gratuitos na página de visão geral da assinatura, isso indicará que eles não estão disponíveis para a assinatura.
