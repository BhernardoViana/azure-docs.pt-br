# <a name="introduction-to-the-azure-ad-connect-synchronization-service-manager-ui"></a>Introdução à interface do usuário do Synchronization Service Manager do Azure AD Connect

![Synchronization Service Manager](./media/how-to-connect-sync-service-manager-ui/ssmui.png)

A interface do usuário do **Synchronization Service Manager** é usada para configurar aspectos mais avançados do mecanismo de sincronização e para ver os aspectos operacionais do serviço.

A interface do usuário do **Synchronization Service Manager** é iniciada no menu Iniciar. Ela é chamada **Synchronization Service** e pode ser encontrada no grupo **Azure AD Connect**.  
![Synchronization Service Manager](./media/how-to-connect-sync-service-manager-ui/startmenu.png)

Clique nos links na parte superior deste tópico para saber mais sobre as diferentes guias da interface do usuário.
