# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Criar, listar ou excluir uma Identidade do Serviço Gerenciado (MSI) atribuída a um usuário usando o Portal do Azure

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

A Identidade de Serviço Gerenciado fornece aos serviços do Azure uma identidade gerenciada no Azure Active Directory. Você pode usar essa identidade para autenticar em qualquer serviço que seja compatível com a autenticação do Azure Active Directory sem precisar ter as credenciais no seu código. 

Neste momento, não é possível criar, listar ou excluir uma identidade gerenciada atribuída a um usuário usando o Portal do Azure.  Procure novamente por atualizações.
