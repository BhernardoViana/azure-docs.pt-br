# <a name="security-info-preview-overview"></a>Visão geral das informações de segurança (versão prévia)

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-security-info.md)]

>[!Important]
>Este conteúdo destina-se aos usuários. Se você for um administrador, é possível encontrar mais informações sobre como configurar e gerenciar seu ambiente do Azure Active Directory (Azure AD) na [Documentação do Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

Configure a verificação em duas etapas e informações de redefinição de senha em **Informações de segurança**. Depois de configurar as informações de segurança inicialmente, você não precisará fazê-lo novamente. No entanto, é possível editar as informações e alterar o método padrão a qualquer momento.

Você pode adicionar um dos métodos abaixo, incluindo:

|Artigo |DESCRIÇÃO |
|------|------------|
|[Configurar informações de segurança para usar chamadas telefônicas](security-info-setup-phone-number.md)| Descreve como configurar o número do telefone como método.|
|[Configurar informações de segurança para usar mensagens de texto](security-info-setup-text-msg.md)|Descreve como configurar o número do telefone como método.|
|[Configurar informações de segurança para usar um aplicativo autenticador](security-info-setup-auth-app.md)|Descreve como configurar um aplicativo autenticador como método.|
|[Configurar informações de segurança para usar email](security-info-setup-email.md)|Descreve como configurar o endereço de email para ajudá-lo a redefinir sua senha.|
|[Configurar informações de segurança para usar perguntas de segurança predefinidas](security-info-setup-questions.md)|Descreve como usar perguntas de segurança para ajudá-lo a redefinir sua senha.|
|[Gerenciar senhas de aplicativo](security-info-app-passwords.md)|Descreve como configurar senhas de aplicativo usando informações de segurança.|
|[Gerenciar suas informações de segurança](security-info-manage-settings.md)|Descreve como atualizar informações de segurança e trabalhar com suas senhas de aplicativo.|
|[Como entrar](user-help-sign-in.md)|Descreve como entrar usando seu método especificado.|
