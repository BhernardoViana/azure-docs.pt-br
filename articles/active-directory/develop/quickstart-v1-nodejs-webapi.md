# <a name="quickstart-secure-a-web-api-with-azure-active-directory"></a>Início Rápido: proteger uma API Web com o Azure Active Directory

[!INCLUDE [active-directory-develop-applies-v1](../../../includes/active-directory-develop-applies-v1.md)]

A [plataforma de identidade da Microsoft](v2-overview.md) é uma evolução da plataforma de desenvolvedor do Azure AD (Azure Active Directory). Ela permite que os desenvolvedores criem aplicativos que se conectam a todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, tais como o Microsoft Graph ou APIs que os desenvolvedores criaram.

## <a name="next-steps"></a>Próximas etapas

Consulte o [Exemplo Web do Azure Active Directory OIDC Node.js](https://github.com/azureadquickstarts/appmodelv2-webapp-openidconnect-nodejs) que demonstra como configurar a autenticação do OpenId Connect com o Azure AD v2.0 em um aplicativo Web criado usando o Node.js com o Express. Ele foi projetado para ser executado em qualquer plataforma.
