# <a name="toggle-live-chat"></a>Alternar chat ao vivo

Este artigo de instruções mostra como ativar e desativar o bate-papo ao vivo em seu aplicativo IoT Central. Você pode usar o chat ao vivo para acessar o suporte técnico.

> [!NOTE]
> A opção de bate-papo está disponível somente para assinaturas de avaliação.

## <a name="chat-with-us"></a>Converse conosco no chat

Para obter suporte técnico, abra o aplicativo e selecione sua Central da IoT **entre em contato conosco**.

![Converse conosco no chat](media/howto-show-hide-chat/chat-with-us.png)

Você pode digitar uma pergunta, conforme mostrado na seguinte captura de tela:

![Formulário "Configurar detalhes do dispositivo" com detalhes para configurações e propriedades](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Ocultar bate-papo

Para ocultar o bate-papo, escolha **ocultar bate-papo** na **ajudar** painel:

 ![Ocultar bate-papo](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Habilitar o bate-papo

Para mostrar o bate-papo, escolha **Mostrar bate-papo** na **ajudar** painel:

 ![Mostrar bate-papo](media/howto-show-hide-chat/show-chat.png)
