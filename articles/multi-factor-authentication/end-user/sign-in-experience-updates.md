# <a name="updates-to-the-sign-in-experience-for-online-services"></a>Atualizações para a experiência de logon para serviços online

Estamos atualizando nossa experiência de logon para serviços online da Microsoft, como Office 365 e Azure. Enquanto isso é apenas uma alteração visual, você observará que as telas parecerão menos confusas e mais simples, tornando mais fácil para que você possa se concentrar no que você precisa fazer – entrar.

## <a name="current-ui-versus-new-ui"></a>Interface do usuário atual versus a nova interface do usuário

Confira a interface do usuário existente em comparação à nova interface do usuário. A nova interface do usuário é mais simplificada e foco, com uma aparência mais atualizada.

![experiência de logon do Microsoft antiga](media/microsoft-sign-in-experience-old.jpg)

![nova experiência de logon do Microsoft](media/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>Quando a nova interface do usuário será exibida?

Aqui está um cronograma de quando esperar que essa alteração ocorra:

| Período de tempo | Alterar |
|-----------|--------|
| Início de maio | Você começará a ver um banner de notificação em sua página de entrada existente. Aqui você pode saber mais e preparar para as alterações. |
| Meio de maio   | As alterações da faixa de notificação incluem um link para visualizar a interface do usuário atualizada. Na visualização, você ainda pode usar um link para alternar novamente para a experiência existente. |
| Meio de junho  | Interface de usuário de entrada atualizada está disponível e todos padrões para a nova experiência de interface do usuário. |

## <a name="related-info"></a>Informações relacionadas

Se você for um administrador, você pode encontrar mais informações sobre essas atualizações com base no produto.

- **Office 365**. Obter mais detalhes sobre as atualizações do [Office 365 Message Center](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail). Você deve entrar usando a autenticação de administrador para acessar este site.

- **Outros serviços online**. Obtenha mais detalhes sobre as atualizações lendo essa [postagem de blog](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/).
