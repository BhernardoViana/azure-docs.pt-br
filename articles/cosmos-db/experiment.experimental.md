# <a name="azure-cosmos-db-documentation-experiment"></a>Experimento de documentação do Azure Cosmos DB

O Azure Cosmos DB é um banco de dados multimodelo distribuído globalmente. Como você pode fazer muito com Azure Cosmos DB - desde criar um gráfico para a criação de um banco de dados de documento ou migrar seus dados do armazenamento de tabela do Azure, Cassandra ou MongoDB - organizar a documentação pode ser um desafio. Portanto, gostaríamos de sua ajuda.

Estamos testando a organização do sumário do Azure Cosmos DB e a [página de aterrissagem](https://docs.microsoft.com/en-us/azure/cosmos-db/), conforme mostrado nas imagens a seguir, e gostaríamos de receber seus comentários sobre essas alterações propostas.

## <a name="to-load-both-views"></a>Para carregar os dois modos de exibição

1. Para abrir os dois modos de exibição lado a lado, navegue até a [página de aterrissagem](https://docs.microsoft.com/en-us/azure/cosmos-db/) do seu navegador favorito. 
2. Use as imagens anteriores para determinar qual versão você vê, original ou experimental. 
3. Abra uma janela de navegador inprivate ou incognito e carregue a [página de aterrissagem](https://docs.microsoft.com/en-us/azure/cosmos-db/) na nova janela do navegador e usar as imagens novamente para determinar qual versão você vê. 

   Se você não tiver os dois modos de exibição presentes, feche e reabra a nova janela inprivate ou incognito e carregue a [página de aterrissagem](https://docs.microsoft.com/en-us/azure/cosmos-db/) novamente. Pode levar algumas tentativas para obter as versões carregadas em janelas separadas. Atualizando a janela incognito não carregará a outra versão, você precisa abrir uma nova janela. 

4. Você pode redimensionar as janelas para corrigir lado a lado, ou alternar entre modos de exibição para comparar. 

5. Depois que você teve a chance de se familiarizar com a organização e o layout da experiência de avaliação, clique no botão **experiências de pesquisa** abaixo para fornecer comentários. Envie o formulário antes da meia-noite de 4 de abril de 2018 para que os comentários possam ser examinados e incorporados.
