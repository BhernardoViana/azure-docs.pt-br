# <a name="azure-cosmos-db-whitepapers"></a>White papers do Azure Cosmos DB 

Os white papers permitem que você explore os conceitos de banco de dados do Azure Cosmos DB em um nível mais profundo. Este artigo fornece uma lista de white papers disponíveis para o banco de dados do Azure Cosmos DB.

| **Whitepaper** | **Descrição** |
| --- | --- |
| [Custo total de propriedade de um serviço de nuvem do banco de dados NoSQL](https://aka.ms/cosmosdb-tco-paper) |Este documento compara o custo financeiro total de propriedade e de software em configurações de implantação diferentes, como um banco de dados local, banco de dados usando a infraestrutura de nuvem. Este documento também fornece as estimativas de custo Total de propriedade para tipos de uso de alta, média e baixa |
|[Indexação independente de esquema com Azure Cosmos DB](http://www.vldb.org/pvldb/vol8/p1668-shukla.pdf) | Este documento descreve o subsistema de indexação do Azure Cosmos DB. Este documento inclui recursos Azure Cosmos DB como representação do documento, a linguagem de consulta, abordagem de indexação de documentos, suporte de índice principal e experiências de produção anterior.|
| [Azure Cosmos DB e dados pessoais](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers)| Este documento fornece orientação para os clientes do Azure Cosmos DB gerenciando um banco de dados baseado em nuvem, um banco de dados local, ou ambos, e que precisam garantir que os dados pessoais em seus sistemas de banco de dados sejam tratados e protegidos de acordo com as regras atuais. |
