# <a name="contributing-to-microsoft-azure-documentation"></a>Contribuindo para a documentação do Microsoft Azure

> Entre 5 e 9 de março de 2019, o repositório MicrosoftDocs/azure-docs passará por uma migração do seu mecanismo de markdown atual para o mecanismo de markdown [Markdig](https://github.com/lunet-io/markdig). Durante esse período, a capacidade de suspender a mesclagem de solicitações de pull públicas será desativada para que a migração seja concluída.

Agradecemos a sua contribuição para contribuir com a documentação do Microsoft Azure.

Este guia aborda alguns tópicos gerais em torno de contribuição e refere-se limite a nossa [guia do Colaborador](https://docs.microsoft.com/contribute) para obter mais explicações, onde relevante.

## <a name="code-of-conduct"></a>Código de Conduta

Este projeto adotou o [Código de Conduta Microsoft Open Source](https://opensource.microsoft.com/codeofconduct/).
Para obter mais informações, consulte o [código de conduta perguntas Frequentes](https://opensource.microsoft.com/codeofconduct/faq/), ou entre em contato com [ opencode@microsoft.com ](mailto:opencode@microsoft.com) com quaisquer outras perguntas ou comentários.

## <a name="how-can-i-contribute"></a>Como posso contribuir?

Há várias maneiras de contribuir com a documentação, examine as seções a seguir para descobrir qual delas é a certa para você.

### <a name="reporting-bugs-and-suggesting-enhancements"></a>Relatório de Bugs e sugerindo melhorias

Use a ferramenta de comentários na parte inferior de qualquer artigo para enviar bugs e sugestões.

![Ferramenta de comentários](media/feedback-tool.png)

### <a name="editing-in-github"></a>Editar no GitHub

Siga as orientações para [rápido edita a documentos existentes](https://docs.microsoft.com/contribute/#quick-edits-to-existing-documents) em nosso guia de colaboradores.

### <a name="pull-request"></a>Solicitação Pull

Examinar a orientação do [solicitações Pull](https://docs.microsoft.com/contribute/how-to-write-workflows-major#pull-request-processing) em nosso guia de colaboradores.
